# Claudit Screenshots

This directory will contain screenshots for the Claudit app page.

## Screenshots Needed

Add the following screenshot files to this directory:

1. **menu-bar.png** - Menu bar quick view showing quota status and cost summary
2. **dashboard-overview.png** - Overview tab with daily cost chart and model distribution
3. **dashboard-projects.png** - Projects tab with time range filtering and search
4. **dashboard-models.png** - Models tab showing detailed cost breakdown per model
5. **dashboard-efficiency.png** - Efficiency tab with cache metrics and recommendations
6. **settings.png** - Settings panel with API key and pricing configuration

## Screenshot Guidelines

- **Format**: PNG
- **Size**:
  - Menu bar: ~400-600px wide (small screenshot)
  - Dashboard: ~1000-1200px wide (full screenshots)
- **Quality**: Compressed but high quality (~200-300KB per screenshot)
- **Styling**: Match existing app screenshots (clean, professional)

## App Icon

Place the app icon at: `/assets/images/app-icons/claudit-icon.png`
- **Size**: 256x256px or larger
- **Format**: PNG with transparency
- **Design**: Match Claudit app icon design
